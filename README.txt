BEFORE USING SCATTLAB2D:

1) edit addpath_scattlab2d.m and set the path to scattlab2d

2) launch addpath_scattlab2d from matlab shell

3) [optional] : add the following two lines to your startup.m file
so that matlab does the addpath automatically when it starts

addpath /path/to/scattlab2d;
addpath_scattlab2d;
