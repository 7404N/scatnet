Usage

[x_phi, x_psi] = wavelet_2d(x, filters, options) whose help is given in wavelet_2d.

Description

It is possible to create some wavelet filters with wavelet_factory_2d for instance. The filters size have to be adapted to the size of the input signal .

x = lena;
filters = morlet_filter_bank_2d(size(x));
[x_phi,x_psi]=wavelet_2d(lena,filters);
figure,

subplot(2,1,1)
imagesc(imag(x_psi.signal{1}))
axis off
title('Imaginary part of the first wavelet transform coefficient');
subplot(2,1,2)
imagesc(imag(x_psi.signal{1}))
axis off
title('Real part of the first wavelet transform coefficient');

Options

Several options are available with wavelet_2d that allow the user to change the output or the way to compute the output.

  • options.x_resolution = 0 changes the resolution on which the wavelet transform is computed by a factor 2.
  • options.precision = 'single' allows to compute either with 'double' or 'single'.
  • options.oversampling = 1 oversamples the signal by a factor 2. £ options.psi_mask = [1 ... 1] is of a size the number of filters and allows to compute the signal only when the corresponding wavelet has a '1' in its psi_mask.
